Einstellungen

🗄️ Supabase-Verbindung einrichten

  1. Gehe auf supabase.com und erstelle einen kostenlosen Account.
  2. Klicke auf "New project", wähle einen Namen (z.B. latein-klasse) und ein Passwort.
  3. Gehe in deinem Projekt zu Settings → API. Kopiere Project URL und anon public key.
  4. Gehe zu SQL Editor und führe das SQL-Script aus dem Abschnitt unten aus.
  5. Gib die Werte unten ein und speichere.

📋 SQL-Setup Script

Kopiere dieses Script und führe es im Supabase SQL Editor aus:

-- Schüler-Tabelle
CREATE TABLE IF NOT EXISTS students (
  id uuid DEFAULT gen_random_uuid() PRIMARY KEY,
  name text NOT NULL,
  code text UNIQUE NOT NULL,
  task_id text,
  created_at timestamptz DEFAULT now()
);

-- Ergebnisse-Tabelle
CREATE TABLE IF NOT EXISTS results (
  id uuid DEFAULT gen_random_uuid() PRIMARY KEY,
  student_id uuid REFERENCES students(id) ON DELETE CASCADE,
  task_id text NOT NULL,
  answers jsonb,
  tries_used integer DEFAULT 0,
  tries_left integer DEFAULT 3,
  is_solved boolean DEFAULT false,
  submitted_at timestamptz DEFAULT now(),
  teacher_comment text,
  updated_at timestamptz DEFAULT now()
);

-- Row Level Security (alle Zugriffe erlaubt, da wir self-hosted Auth nutzen)
ALTER TABLE students ENABLE ROW LEVEL SECURITY;
ALTER TABLE results ENABLE ROW LEVEL SECURITY;

CREATE POLICY "allow_all_students" ON students FOR ALL USING (true) WITH CHECK (true);
CREATE POLICY "allow_all_results" ON results FOR ALL USING (true) WITH CHECK (true);
